## 4. 理想 CIR 到可观测 CIR 的转换

### 4.1 理论方法

从 RT 获得的理想 δ CIR：
```
h_ideal(τ) = Σ α_l · δ(τ - τ_l)
```

经过系统带宽限制后的可观测 CIR：
```
h_observed(τ) = h_ideal(τ) * p(τ)
```
其中 p(τ) 是测量系统的等效脉冲响应（由带宽 BW 和窗函数 W(f) 决定）。

### 4.2 频域方法（IFFT）

| 步骤 | 说明 |
|------|------|
| 1. 生成 CFR | H(f_i) = Σ α_l · exp(-j2πf_i·τ_l) |
| 2. 加窗 | H_windowed(f_i) = H(f_i) · W(f_i) |
| 3. IFFT | h_observed(τ_k) = IFFT{H_windowed(f_i)} |
| 4. 时延分辨率 | Δτ = 1 / BW |
| 5. 最大无模糊时延 | τ_max = N_pts / BW |

### 4.3 窗函数选择

| 窗函数 | 主瓣宽度 | 旁瓣抑制 | 适用场景 |
|--------|---------|---------|---------|
| 矩形窗 | 1/BW | -13 dB | 最高分辨率 |
| Hann 窗 | 2/BW | -31 dB | 通用 |
| Hamming 窗 | 2/BW | -43 dB | 强旁瓣抑制 |

## 6. 本论文的仿真→实测对照方案

### 推荐流程

```
几何场景建模 (OBJ) → 材质赋值 → RT仿真 → precise路径集
    ↓
多频点扫频 (frequency_sweep) → H(f_i) → 加窗 → IFFT → h_observed(τ)
    ↓
与3GPP TR 38.901 InH-Office 统计参数对照
    ↓
对比指标: path_loss_RMSE, delay_spread, K_factor, XPR
```

### 论文可用的三层替代验证

1. **解析对照**（最可靠）：LOS Friis、单反射 Fresnel、Snell 折射——与手算解析值对照
2. **Sionna RT 对照**：同等场景用 Sionna RT 生成结果，对比路径损耗/PDP/APS
3. **3GPP 统计对照**：与 3GPP TR 38.901 InH-Office 的路损/DS/AS/K 因子统计值对比
