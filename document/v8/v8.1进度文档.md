## 一、项目总览

RT 电磁传播仿真系统 — C++20 六模块分层架构，Precise IM + SBR Monte Carlo 双模式。

```
app/                   主流水线 (RtPipeline + RtRealChainRunner)
core/search/           寻径引擎 (SearchEngine + SbrEngine + 三展开器)
core/em/               电磁计算 (14个 Apply*/Build* 函数)
core/query/            空间加速 (SceneQuery + BVH + ISceneAccelerator)
core/scene/ + path/    场景 + 路径数据结构
core/common/ + antenna/ 基础设施 + 天线模块
preprocess/            场景预处理 (导入→拓扑→加速→缓存)
```

## 三、当前 coverage 模式数据流

```
配置: meeting_412.json
  scenes: 6775 faces, 6864 Rx grid (0.1m step)
  SBR: 2M rays, depth 8 (refl=4, trans=2, diff=0)
  store_paths: true
  solver_mode: Coverage
  enable_wedge_build: false (6775面楔边构建待修复)

管线:
  config→import→topo→BVH→SceneQuery
    ↓
  SBR 寻径 (2M rays, CPU 28核 OpenMP)
    ├─ BVH traversal (QueryClosestFaceHitFast + AVX2 SIMD)
    ├─ Rx spatial grid (密集3D数组, O(1)直接索引)
    ├─ Fresnel cached (thread_local, O(1)查表)
    └─ 每Rx命中 → 构建 GeometricPath (PathSignature去重)
    ↓
  SBR→Precise EM (逐路径)
    ├─ InitializeTxField (天线增益+极化+姿态)
    ├─ ApplyFreeSpaceSegment (距离/时延/相位/介质衰减)
    ├─ ApplyReflectionInteraction (Fresnel TE/TM + Jones矢量)
    ├─ ApplyTransmissionInteraction (Fresnel τ + 介质α + n_eff)
    └─ FinalizeAtReceiver (FSPL + Rx天线 + 极化)
    ↓
  Per-Rx 相干叠加 |ΣA_i|² → power_linear/dBm
    ↓
  Export: sbr_coverage.json (含 tx_position, rx_positions)
```

### 已知限制

| 项目 | 说明 |
|------|------|
| 绕射 | SBR 保持标量近似 (不进入 Precise EM 链) |
| 透射 MC | 概率选择 (`P=|τ|²`→透射, `P=|Γ|²`→反射) — 非确定性分裂 |
| 接收球近似 | `sphereR=0.1m` — specular 点几何误差 bounded by R |
| 楔边构建 | >1000 面元时 `bad_alloc` — 需空间哈希修复 |
| 双面穿墙透射 | 第二面阻挡反射可见性 |

```

4. 修改 SceneQuery 构造器:
   if (config.acceleration.backend == "GPU_OptiX")
       accelerator_ = std::make_unique<OptiXSceneAccelerator>(scene, config);
```
