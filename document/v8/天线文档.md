## 2. 已实现功能

### 2.1 增益方向图 (Gain Pattern)

**数据格式** (CSV):
```csv
Theta[deg],Phi[deg],Gain_dBi
0,0,2.15
10,0,2.10
...
```

**坐标系约定** (Y-up):
- θ=0°: +Y方向 (天顶, boresight默认)
- θ=90°: 水平面
- θ=180°: -Y方向 (天底)
- φ=0°: +X方向
- φ=90°: +Z方向

**插值算法**: θ-φ双线性插值。`AntennaPattern::QueryGainDBi(thetaDeg, phiDeg)`

**注入链路**:
- Tx: `InitializeTxField` — 发射方向 → 天线局部球坐标 → 查增益 → `amplitude *= sqrt(G_lin)`
- Rx: `FinalizeAtReceiver` — 到达方向 → 天线局部球坐标 → 查增益 → 总功率 *= G_lin

**配置示例**:
```json
"antenna": {
    "source_type": "Pattern",
    "pattern_file": "materials/hfss_dipole_3p5GHz.csv",
    "polarization_file": ""
}
```

### 2.2 天线姿态 (Antenna Orientation) — v8 新增

**配置字段**:
```
forward_x, forward_y, forward_z  — 天线主瓣指向 (boresight), 世界坐标
up_x, up_y, up_z                — 天线"上"方向, 世界坐标 (定义极化参考面)
```

天线局部坐标系自动计算: `right = forward × up`（归一化正交）

**坐标变换**: `WorldToAntennaSpherical(worldDir, forward, right, up)` → (θ_antenna, φ_antenna)

**效果**:
- 方向图查询使用天线局部球坐标，自动适配天线旋转
- 主瓣始终对应 θ=0°（天线局部forward方向）
- 示例：天线指向+Y时，+Y方向的发射对应θ=0°（主瓣最大增益）

**配置示例** (天线指向东北上方):
```json
"antenna": {
    "forward_x": 1.0, "forward_y": 1.0, "forward_z": 0.5,
    "up_x": 0.0, "up_y": 0.0, "up_z": 1.0
}
```

### 2.3 逐角度极化方向图 (Per-Angle Polarization) — v8 新增

**数据格式** (CSV, Ludwig-3定义):
```csv
Theta[deg],Phi[deg],Gain_dBi,PolTheta_re,PolTheta_im,PolPhi_re,PolPhi_im
0,0,2.15,1.0,0.0,0.0,0.0
...
```

| 列 | 含义 | 来源 |
|----|------|------|
| Theta[deg] | 天线局部θ角 | — |
| Phi[deg] | 天线局部φ角 | — |
| Gain_dBi | 该方向增益 | HFSS/CST远场报告 |
| PolTheta_re/im | θ̂方向复极化分量 (实部/虚部) | HFSS rEtheta |
| PolPhi_re/im | φ̂方向复极化分量 (实部/虚部) | HFSS rEphi |

**Ludwig-3定义**: 极化分量在球坐标基 (θ̂, φ̂) 中表达，独立于天线姿态。总电场 = polTheta·θ̂ + polPhi·φ̂。

**世界坐标重建**: `AntennaSphericalBasisToWorld()` 将 (θ̂, φ̂) 转换到世界坐标，然后 `E_world = polTheta·θ̂_world + polPhi·φ̂_world`。

**HFSS/CST 导出指南**:
1. HFSS: `Radiation → Far Field → Export → Ludwig-3 (θ̂, φ̂) → rEtheta, rEphi (real+imag)`
2. CST: `Farfield Plot → Export → ASCII → Theta/Phi components → Real+Imag parts`
3. 转换为CSV格式: `θ, φ, Gain_dBi, Re{rEtheta}, Im{rEtheta}, Re{rEphi}, Im{rEphi}`
4. 归一化: 确保 (|rEtheta|² + |rEphi|²) 在各方向归一化到1（增益已单独处理）

**固定极化兼容** (v7): 若 `polarization_file` 内容为单行3个浮点数（非CSV表头），自动回退为固定极化矢量：
```
0.0 1.0 0.0    ← 表示 polarization_vector = (0,1,0), Y-up线偏振
```

**注入链路**:
```
InitializeTxField:
  ├─ 若 polarization_loaded:
  │   ├─ QueryPolarization(θ, φ) → (polΘ_re, polΘ_im, polΦ_re, polΦ_im)
  │   ├─ AntennaSphericalBasisToWorld → θ̂_world, φ̂_world
  │   └─ Jones = polΘ·θ̂ + polΦ·φ̂ (世界坐标, 归一化)
  └─ 否则:
      └─ polarization_vector = 固定极化矢量 (默认Y-up)
```

### 3.2 天线阵列 (P2)

**需求**: MIMO 阵列合成。开题报告要求"天线联合建模"。

**当前状态**: `AntennaArrayModel` 结构已定义但未接入主流程。

**实现方案**:
1. 对每阵元独立运行几何寻径（IM路径共享可用synthetic array）
2. 各阵元独立EM求解（仅极化/增益/相位不同）
3. 相干叠加得到阵列响应
4. 相控阵波束赋形: 附加阵元间相位差

### 3.3 相位中心偏移 (P3)

`AntennaModel::phase_center_offset_m` 已定义但未用于路径几何。应在几何寻径前将Tx/Rx位置偏移相位中心。

### 3.4 天线时延响应 (P3)

开题报告要求"时延-极化-旋转联合建模"。天线群时延（随频率的相位响应）可用于超宽带信道建模。

## 6. HFSS/CST 导出脚本参考

### Python脚本: HFSS远场 → RT极化CSV

```python
"""
Convert HFSS far-field export to RT polarization CSV format.

HFSS export steps:
  1. Radiation → Insert Far Field Setup → Infinite Sphere
  2. Results → Create Far Field Report → rEtheta, rEphi (Real+Imag)
  3. Export → CSV (theta, phi, re_rEtheta, im_rEtheta, re_rEphi, im_rEphi, gain_dB)
"""
import numpy as np

def hfss_to_rt_polarization(hfss_csv, output_csv):
    """Convert HFSS export to RT format."""
    data = np.loadtxt(hfss_csv, delimiter=',', skiprows=1)
    # Expected HFSS columns: theta, phi, reEtheta, imEtheta, reEphi, imEphi, gain_dB
    theta = data[:, 0]
    phi   = data[:, 1]
    gain  = data[:, 6]  # or compute from |E|^2
    
    pt_re = data[:, 2]  # Re{rEtheta}
    pt_im = data[:, 3]  # Im{rEtheta}
    pp_re = data[:, 4]  # Re{rEphi}
    pp_im = data[:, 5]  # Im{rEphi}
    
    # Normalize polarization to unit magnitude at each point
    mag = np.sqrt(pt_re**2 + pt_im**2 + pp_re**2 + pp_im**2)
    mag[mag < 1e-15] = 1.0
    pt_re /= mag; pt_im /= mag; pp_re /= mag; pp_im /= mag
    
    with open(output_csv, 'w') as f:
        f.write('Theta[deg],Phi[deg],Gain_dBi,PolTheta_re,PolTheta_im,PolPhi_re,PolPhi_im\n')
        for i in range(len(theta)):
            f.write(f'{theta[i]:.1f},{phi[i]:.1f},{gain[i]:.3f},{pt_re[i]:.6f},{pt_im[i]:.6f},{pp_re[i]:.6f},{pp_im[i]:.6f}\n')
```
