## B.1 实验设计

### 扫盲层

天线指向不同方向时，它对不同来波方向的增益不同。就像你转动身体时，耳朵对不同方向的声音敏感度会变化——正对声源时听得最清楚，背对时几乎听不到。**E2 实验把 Rx 天线从 0° 转到 180°，观察接收功率和信道指标如何随指向变化。**

### 学术层

天线 yaw 旋转改变 Rx 的 `forward` 方向矢量（绕 Z 轴旋转），从而改变 `WorldToAntennaSpherical` 计算出的局部角 (θ, φ)。在给定固定的来波方向分布（由场景几何和 Tx 位置决定）下，yaw 变化等效于在天线方向图上沿 φ 方向"扫描"。

本实验使用 **patch 天线**（定向，~6-8 dBi 主瓣，明显的方向性），Tx 固定指向 +X，Rx yaw 从 0° 到 180°。

**Rx 局部坐标定义**：
- forward = [cos(yaw), sin(yaw), 0]
- up = [0, 0, 1]
- right = forward × up = [-sin(yaw), cos(yaw), 0]

**Tx→Rx 几何关系**：
- Tx 位置：(10, 1.5, −10)
- Rx 位置：(7, 1.5, −12)
- 入射方向（从 Rx 看向 Tx）：(3, 0, 2) → 水平面内 φ ≈ 33.7°, θ ≈ 90°

## B.3 关键发现

### 发现 1：接收功率跨越 200 倍（23 dB）

```
最低：Yaw 60° — 5.64E-10 W (−62.5 dBm)
最高：Yaw 180° — 1.11E-07 W (−39.6 dBm)
动态范围：23 dB
```

**解释**：
- 当 yaw=180° 时，Rx forward 指向 −X 方向，Rx 的"正前方"正好对准 Tx 方向（入射方向 φ≈33.7° 投影到天线局部坐标接近 boresight），patch 主瓣增益最大。
- 当 yaw=60° 时，Rx forward 指向 (0.5, 0.866, 0)，boresight 偏离 Tx 方向 ~90°，patch 增益最小（旁瓣/背瓣区域）。
- **这是天线方向图直接调制接收功率的量化证据。**

### 发现 2：RMS 延迟扩展随 yaw 显著变化

```
最低：Yaw 180° — 6.8 ns
最高：Yaw 90° — 15.6 ns
变化范围：2.3×
```

**解释**：
- 当 Rx 主瓣指向 Tx 方向（yaw=150°–180°）时，Rx 优先接收来自 Tx 方向的短延迟路径（主要是 LOS 和低阶反射），抑制来自其他方向的长延迟路径 → RMS-DS 低。
- 当 Rx 主瓣偏离 Tx 方向（yaw=60°–90°）时，Rx 抑制了短延迟路径（因为它们来自 boresight 以外的方向），反而选择性接收了来自旁瓣方向的远距离反射 → RMS-DS 高。

### 发现 3：功率最大时 RMS 最小，功率最小时 RMS 最大

这是天线方向性在角度-延迟域中作用的直接体现：

**主瓣对准 Tx → 高增益收集短延迟路径 → 高功率 + 低 RMS**  
**主瓣偏离 Tx → 低增益抑制短延迟路径，旁瓣收集长延迟路径 → 低功率 + 高 RMS**

### 发现 4：K-factor 从极端负值改善至 −36 dB

K-factor 从 yaw=0° 的 −58.7 dB 改善至 yaw=90° 的 −35.8 dB。当 Rx 主瓣不对准 Tx 时，LOS（来自 Tx 方向）被衰减，但大量 NLOS 路径来自旁瓣方向，导致 NLOS 功率相对更高。当 Rx 主瓣对准更强的反射体方向时（yaw=90° 时 forward 指向 +Y），NLOS 被选择性增强，K-factor 恶化程度略减。

## B.4 对论文结论的支撑

E2 实验直接证明了：**天线三维姿态（即使是单轴旋转）对 P2P 信道指标产生 20+ dB 功率调制和 2×+ RMS 延迟扩展变化**。

这对系统设计的含义：在室内环境中，终端设备的天线指向对链路质量有决定性影响。论文中应强调"天线姿态分集"作为提升链路可靠性的潜在技术方向。
