# V11.6 全链路修复与自测报告

> 日期：2026-06-29  
> 范围：P2P SBR 候选发现、物理路径精修、路径级 EM、独立 UTD、固定增益宽带及基础结果导出。

## 1. 一句话结论

修复前，程序把接收球附近的采样射线直接送入 EM；修复后，采样射线只负责报告“这里可能有一条路径”，程序会重新求出严格满足反射/Snell/UTD 几何的物理路径，再计算复场、功率、极化和 CIR。

## 2. 为什么这样修

成熟 SBR 的共同思想不是无限信任某一根离散射线，而是：SBR 负责高效发现候选，精确几何方法负责恢复真路径，物理残差负责验收。这样做能同时保留 SBR 的速度和确定性镜面路径的物理稳定性。

接收球半径和射线数现在主要影响“能否发现”和“有多少候选支持”，不再直接改变一条已发现镜面径的 Tx 功率。这个区分是可解释仿真的基础。

## 3. 模块级结果

| 模块 | 修复前问题 | 本轮处理 | 可观察结果 |
|---|---|---|---|
| SBR 接收 | Rx 点被改成球心，但末段长度仍取射线最近点 | 候选拓扑后重新求全部交互点并连接 Rx 中心 | 分段长度严格求和，总时延稳定 |
| 面元去重 | 同一墙面的不同三角形/采样点被当成不同路径 | 建立相连共面物理面片，按交互拓扑聚类 | 路径数代表物理径，不代表命中射线数 |
| 反射验证 | 单反射残差被写死为 0 | 用出射方向与镜面方向的向量差验收 | 怪径不能再靠结构完整混入 EM |
| 透射验证 | 原始 SBR 交点未针对 Rx 重新满足 Snell | 约束 LM 联立求解 R/T 节点 | 真实透射例残差约 `1.26e-9` |
| 初始场 | `sampling_weight` 乘入 Tx 功率 | Tx 功率与采样支持度分离 | 射线更多不会机械增加/减少单径功率 |
| R/T EM | 只查询单侧材料，输入输出 TM 共用一套基 | 双侧复介质 Fresnel，分别建立输入/输出基 | 无耗 `R+T=1`，输出场保持横向 |
| UTD | 缺球面扩散补偿、输入场投影到输出基、外角和凸凹分类错误 | 独立 soft/hard 基、扩散补偿、有符号凸凹分类、逐路径外部传播域检查 | 只保留可解释的实体凸边绕射，规范楔角与 412 场景均产生有限功率 |
| CFR | 中心频率相位被重复乘一次传播时延 | 使用频偏 `f-fc` 旋转中心频率复系数 | 中心频点严格恢复窄带复信道 |
| Coverage | 固定 `1 uW` 门槛删除全部正常弱径 | 聚合保留寻径已接受的有效路径 | 412 不再输出 0 功率/0 路径 |

## 4. 输出如何阅读

### `precise_paths.csv`

每行是一条最终物理路径。重点字段：

- `power_linear` / `power_dBm`：Rx 天线投影后的单径接收功率。
- `channel_h_real/im`：相对于 `sqrt(Ptx)` 的无量纲复信道系数。
- `incident_E*_vpm`：Rx 位置、天线投影前的物理复电场，单位 V/m。
- `candidate_support_count`：多少条发射射线发现了该拓扑，只用于收敛诊断。
- `geometry_refined`、`refinement_method`、`geometry_residual`：路径是否精修、怎么精修、物理残差多大。

### `path_node_fields.csv`

每行是某条路径的一个节点：

- `in_E*_sqrtW`：传播到节点、执行交互前的复功率波。
- `out_E*_sqrtW`：反射/透射/绕射后的复功率波。
- `active_medium_id`：交互完成后继续传播所处介质。
- `cumulative_length_m`、`delay_s`：到该节点为止的长度和介质时延。

通俗地说，沿同一 `path_id` 从 node 0 往后读，就能看到极化方向、振幅和相位经过每一面墙或棱边后怎样变化。

## 6. 剩余风险和能力边界

1. 当前绕射仍限定为独立 `Tx-D-Rx`；R/T/D 混合径、多次绕射和漫反射未实现。
2. UTD 首版按 PEC soft/hard 系数验收；有限电导楔面的阻抗修正仍需单独实现和对标。
3. `frequency_sweep_em` 仍明确返回 unsupported；已闭环的是固定路径增益的窄相对带宽模式。
4. 物理面片依赖共享拓扑边。OBJ 若把本应连续的同一表面拆成重复顶点，需要导入阶段焊接或显式 patch ID。
5. 含透射精修固定候选交互面片和拓扑；高曲率细网格、极端掠入射和临界角附近仍应做专项数值压力测试。
6. 节点级内部量是 `sqrt(W)` 功率波，不是局部绝对 V/m；本轮只在 Rx 端给出严格物理 V/m。若论文需要沿路径任意位置的绝对场强，还需引入逐段波前曲率/散度张量。
7. 接收球仍影响候选发现概率，因此论文实验仍应报告射线数与接收半径收敛，而不能只报告一次运行。
8. 412 OBJ 当前含 469 个退化面；相关 1011 条共享边已从 UTD 候选中拒绝，但源模型仍需清理并重新导出。

这些边界不会影响本轮已经闭环的路径几何、路径级功率、节点 Jones 演化、AoA/AoD 和固定增益 CIR，但决定了后续扩展实验能否被严谨解释。
