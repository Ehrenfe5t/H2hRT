# v11.7 真实材料有限电导 UTD 修复流程

日期：2026-06-29  
状态：实现、构建、自测、端到端验证和全量回归均已完成

## 1. 本步目的

旧绕射链路能够计算 PEC 楔边绕射，但不能完整表达组成楔边的两个真实表面材料。这样会把混凝土、玻璃等介质楔边近似为理想金属，导致绕射振幅、相位和极化缺少材料可解释性。

本步目标是建立以下闭环：

1. 从场景面元保留楔边两侧的真实表面材质。
2. 在当前频率查询各自的介电常数和电导率。
3. 分别计算两侧面的复 TE/TM Fresnel 反射系数。
4. 用 Kouyoumjian-Pathak UTD 四项系数和 Jones 基变换计算矢量绕射场。
5. 保留 Tx 到绕射节点再到 Rx 的复电场演化及诊断量。
6. 通过解析锚点、材料极限、极化约束、端到端场景和全量回归闭环验证。

当前明确不实现反射/透射与绕射混合路径；本步只处理独立面反射/透射路径和独立棱边绕射路径。

## 2. 方法选择及理由

采用有限电导楔边 UTD，而不是继续使用统一 PEC 系数。原因如下：

- UTD 是高频射线框架中处理楔边阴影边界连续性的成熟方法，与现有 SBR/P2P 路径表示一致。
- 两个楔面分别乘以其复 Fresnel TE/TM 系数，可表达材料损耗、材料相位和极化差异。
- 通过二维 Jones 矩阵完成入射边基、面 TE/TM 基和出射边基之间的转换，可避免把三维矢量场错误地当作单一标量。
- 该方法与 NVIDIA Sionna RT 的有限电导楔边实现结构一致，复杂度适合当前硕士论文仿真范围。

主要参考：

- Kouyoumjian and Pathak, *A Uniform Geometrical Theory of Diffraction for an Edge in a Perfectly Conducting Surface*, 1974。
- [Sionna RT Path Solver](https://nvlabs.github.io/sionna/rt/tech-report/S3.html)
- [Sionna RT Electromagnetic Field Computation](https://nvlabs.github.io/sionna/rt/tech-report/A1.html)
- [Sionna RT 开源实现](https://github.com/NVlabs/sionna-rt)

## 3. 理论到代码的映射

### 3.1 材料与 Fresnel 系数

对楔边的 0 面和 n 面分别查询材料参数，并由复介电常数计算空气到材料界面的 TE/TM 复反射系数。两个面的材料、入射角和反射系数独立，不再共用一个 PEC 常数。

若任一面的材质名称为空、数据库不存在或查询失败，该绕射路径直接判为 EM 无效。这里不允许静默回退为 PEC，因为这种回退会制造数值正常但物理错误的结果。

### 3.2 UTD 绕射矩阵

先计算四个 Kouyoumjian-Pathak 项 `d1`、`d2`、`d3`、`d4`，再构造：

```text
D = -(d1 + d2) I
    + W0_out diag(R0_TE d4, R0_TM d4) W0_in
    + Wn_out diag(Rn_TE d3, Rn_TM d3) Wn_in
```

其中 `W` 是 Jones 坐标基旋转矩阵。最终通过入射边基投影、矩阵相乘和出射边基重建得到三维复矢量场。

`phiPrime` 按“绕射点指向源”的方向计算；若误用入射传播方向，UTD 方位角会反向，进而改变四项系数的组合。

### 3.3 UTD 过渡函数修复

最终复核发现旧实现通过截断无穷振荡尾积分计算 UTD 过渡函数。独立数值对照显示，相对误差会从小参数处约 4% 增长到中大参数处 30% 以上。

现改为等价的有限 Fresnel 积分：

```text
F(x) = sqrt(pi*x/2) exp(jx) [(1+j) - 2j conj(C(x)+jS(x))]
```

有限区间使用分段 8 点 Gauss-Legendre 积分，`x > 50` 使用渐近式。该方式不会截断无穷振荡尾部，并用 `F(1)`、`F(10)` 的独立参考值建立数值锚点。

### 3.4 距离、相位与衰减

自由空间传播段负责累积：

- 传播相位 `exp(-j k n d)`；
- 介质振幅衰减 `exp(-alpha d)`；
- 路径长度和时延。

总路径 Friis 振幅因子只在接收端应用一次。UTD 球面扩散使用补偿项：

```text
sqrt((s1+s2)/(s1*s2))
```

这样与接收端统一应用的 `lambda/(4*pi*(s1+s2))` 配合，不会重复计算自由空间扩散损耗。

### 3.5 极化与单位

节点内部场使用复功率波形式，单位标记为 `sqrt(W)`，用于稳定地串联天线增益、交互 Jones 矩阵和相干相位。接收点另外换算并导出真实入射电场 `V/m`。

因此不能把节点 CSV 中的 `sqrt(W)` 数值直接称为局部真实场强。两者都保留，服务于不同分析目的。

## 6. “逐节点电场算对是否就满足要求”的结论

大白话结论：逐节点把相位、振幅和极化算对，是最核心的主干，但单独这一句话还不够。

只有以下五层同时成立，才能认为当前支持范围内的单路径信道仿真满足要求：

1. 路径几何正确：节点位置、交互类型、面法向、楔边和传播方向正确。
2. Tx 初始条件正确：发射功率、方向图、初始 Jones 极化和单位定义正确。
3. 节点交互正确：每次反射、透射或绕射使用正确材料、角度、相位和极化基。
4. 传播标度正确：传播相位、介质损耗、自由空间扩散只计算一次。
5. Rx 收尾正确：接收天线共轭投影、增益、`V/m` 换算、路径间相干叠加和输出定义正确。

本次修改已把独立棱边绕射纳入这条主干，并给出节点级可审计证据。因此，在“高频远场、确定性镜面反射/透射、独立凸楔边绕射、窄带频点”这一明确范围内，可以认为基础 EM 链路满足论文仿真的主要要求。

## 7. 仍需保留的适用边界

以下内容不是本次实现错误，但不能在报告中隐去：

- 当前不支持反射/透射与绕射混合路径。
- 当前只接收几何侧筛选后的独立凸楔边；不模拟凹边、共面缝和漫散射绕射。
- 楔面按局部半无限材料界面处理，没有引入墙体厚度对应的多层板绕射系数。
- 当前材料模型按常用非磁性、各向同性介质处理；未使用频变磁导率、各向异性或表面粗糙度。
- 宽带结果目前不是每个频点重新寻径和重新计算材料 UTD。
- SBR 不保证发现数学意义上的全量路径；结果完备性仍由射线密度、接收机制和收敛实验约束。
- 节点内部 `sqrt(W)` 是功率波，不是 `V/m`；真实入射场强在 Rx 结果中单独导出。

这些边界不妨碍当前硕士论文范围内的基础结果使用，但论文和实验报告必须按上述口径说明。
