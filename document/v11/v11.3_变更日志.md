# V11.3 变更日志：论文指标聚合与导出补全

日期：2026-06-27
状态：✅ **已完成**
依据：`v11开发文档.md` §3.1、§3.5、§5 阶段 V11.3、§10.11

## 目标

让一次 P2P 仿真直接产出论文画图所需数据，不再依赖手工从 `precise_paths.json` 临时统计。

---

## 变更清单

### D1: `BuildChannelStatistics` 增强
- **文件**：`core/em/BuildChannelStatistics.cpp`（重写）
- **新增计算**：
  - `rms_delay_spread_s`：功率加权 RMS 延迟扩展 = sqrt(Σ(P_i·(τ_i-τ̄)²) / ΣP_i)
  - `k_factor_dB`：10·log10(P_LOS / P_NLOS)，纯LOS=+300，纯NLOS=-300
  - `effective_path_count`：功率 ≥ 最强径/100（-20 dB 阈值）的路径数
  - `strongest_path_delay_s`：最强路径的延迟
- **保留**：valid_path_count、total_power_linear、strongest_path_power_linear、mean_delay_s、transmission_path_count

### D2: 新增 `BuildXPRStatistics`
- **文件**：新建 `core/em/BuildXPRStatistics.h`、`core/em/BuildXPRStatistics.cpp`
- **输出**：mean、median、P10、P90、power-weighted mean、min、max、排序 CDF 数组
- **排除**：±300 dB 哨兵值（纯 co/cross-pol 路径）
- **注册**：`RT.vcxproj` 添加编译条目

### D3: `BuildAPS` 增强 — 2D AoA 网格
- **文件**：`core/em/BuildAPS.h`（新增重载声明）、`core/em/BuildAPS.cpp`（重写）
- **新增**：`BuildAPS(pathResults, config)` 重载，从 `config.em_solver.aps_theta_bins`/`aps_phi_bins` 读取网格参数
- **输出**：`APSResult.has_2d_grid=true`，`power_grid_linear`（nTheta×nPhi），`power_grid_dB`（-200 dB 地板）
- **保留**：1D entries（向后兼容）

### D4: 新增 `ComputeMEG`
- **文件**：新建 `core/em/ComputeMEG.h`、`core/em/ComputeMEG.cpp`
- **算法**：MEG = Σ(G_Rx(θ,φ)·P_obs(θ,φ)·sinθ) / Σ(P_obs(θ,φ)·sinθ)
- **当前**：Rx 天线为 null 时使用理想全向增益（MEG=1.0=0dB），为后续接入实际 Rx 方向图预留接口
- **注册**：`RT.vcxproj` 添加编译条目

### D5: `BuildAggregateResult` 接线
- **文件**：`app/RtRealChainRunner.cpp`
- **变更**：
  - 新增 `#include BuildXPRStatistics.h / ComputeMEG.h`
  - `BuildAggregateResult` 新增 `const AppConfig&` 参数
  - 调用 `BuildAPS(pathResults, config)` 替代旧 `BuildAPS(pathResults)`
  - 新增 `result.xpr_stats = BuildXPRStatistics(pathResults)`
  - 新增 `ComputeMEG()` 调用（条件：`config.em_solver.compute_meg && aps.has_2d_grid`）
  - 两处调用点传入 `config` 参数

### D6: `ExportChannel` 扩展
- **文件**：`core/result/ExportChannel.cpp`（重写）
- **新增导出文件**（每个 Tx-Rx 链路）：
  | 文件 | 内容 |
  |---|---|
  | `channel/cir.json` / `cir.csv` | CIR taps：delay_s, amplitude_real, amplitude_imag, power_linear |
  | `channel/pdp.json` / `pdp.csv` | PDP taps：delay_s, power_linear, coherent_power, incoherent_power |
  | `channel/aps_2d_grid.json` / `aps_2d_grid.csv` | 2D theta×phi 功率网格（线性 + dB） |
  | `channel/xpr_stats.json` | XPR 统计：mean/median/P10/P90/power-weighted mean/min/max + CDF 数组 |
  | `channel/channel_stats.json` | 信道统计：RMS delay spread, K-factor, effective path count 等 |
  | `channel/meg.json` | MEG：meg_linear, meg_dB |
- **保留**：`channel_summary.json`（扩展版）、宽带导出（条件触发）
