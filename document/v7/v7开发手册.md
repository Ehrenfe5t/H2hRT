## 第1章：系统总览

### 1.1 项目定位

室内 ISAC（通感一体化）多维域信道建模 RT 仿真系统。C++20 + MSVC，六模块分层架构，支持 **precise Image Method**（精确几何寻径）和 **SBR Monte Carlo**（大规模覆盖仿真）双模式。

### 1.2 六模块架构

```
app/                主流水线 (RtPipeline)
  Batch0/1: 配置加载+校验
  Batch2:   场景导入 (OBJ)
  Batch3:   拓扑构建 (Edge/Wedge/BVH)
  Batch4:   查询门面+场景缓存
  Batch5:   搜索引擎 (precise IM)
  A1:       EM求解+导出 (precise)
  SBR:      覆盖仿真 (coverage, 可选)

core/search/        寻径引擎
  SearchEngine:      优先队列 IM 搜索
  ReflectionExpander: 镜像法反射
  TransmissionExpander: Snell 透射
  DiffractionExpander: Fermat+Keller 绕射
  GeometryValidity:  几何合法性检查
  SbrEngine:         SBR Monte Carlo

core/em/            电磁计算
  InitializeTxField: Tx 场初始化
  ApplyFreeSpaceSegment: 自由空间传播
  ApplyReflectionInteraction: Fresnel TE/TM 反射
  ApplyTransmissionInteraction: Fresnel TE/TM 透射
  ApplyDiffractionInteraction: UTD 绕射
  FinalizeAtReceiver: FSPL+Rx天线收敛
  BuildCIR/PDP/APS/ChannelStatistics/ISAC: 后处理

core/query/         空间加速
  SceneQuery: BVH 遍历+三角求交
  RxBVH.h: Rx空间索引 (预留)
  ISceneAccelerator.h: GPU加速接口 (预留)

core/scene/         场景数据结构
  Face, Edge, Wedge, Scene

preprocess/         场景预处理
  import:   OBJImporter
  binding:  MaterialRuleLoader, ResolveFaceDualSideMaterial
  build:    EdgeBuilder, WedgeBuilder, SceneTopologyBuilder
  accel:    FaceBVHBuilder (SAH 16-bin)
  cache:    SceneCache

core/common/        基础设施
  math:     Vec3, Complex, Matrix3x3, MathConstants
  config:   AppConfig, AppConfigJsonCodec, AppConfigValidator
  material: MaterialDatabase (ITU-R P.2040)
  log:      Logger
```

### 1.3 双模式对比

| 维度 | Precise (Image Method) | SBR Coverage |
|------|----------------------|--------------|
| 算法 | 优先队列 IM + 三展开器 | Monte Carlo 射线追踪 |
| 路径数 | 362~1572 (场景相关) | 200万射线 × 1.25M Rx |
| 极化 | Jones 矢量 TE/TM 相干 | 标量 Fresnel |Γ|² |
| 绕射 | UTD 8点 GL 数值积分 | Keller 锥 + UTD 频率校准 |
| 透射 | Snell + Fresnel 相干 | MC 概率选择 + 物理衰减 |
| 并行 | 逐路径 OpenMP | 逐射线 OpenMP 28核 |
| 产出 | 路径级复振幅/时延/AoA/极化 | Rx 网格功率热力图 |

## 第3章：关键算法实现详解

### 3.1 Precise 模式

详见 `v7.2precise.md`。核心流程：

```
SearchEngine::Run()
  └─ 优先级队列 (最佳优先)
       ├─ ReflectionExpander: 镜像法
       ├─ TransmissionExpander: Snell折射
       └─ DiffractionExpander: Fermat+Keller锥

RtRealChainRunner::SolveSinglePathEM()
  └─ for each path node:
       ├─ ApplyFreeSpaceSegment (自由空间传播)
       ├─ ApplyReflectionInteraction (Fresnel TE/TM)
       ├─ ApplyTransmissionInteraction (Fresnel TE/TM透射)
       └─ ApplyDiffractionInteraction (UTD D₁-D₄)
  └─ FinalizeAtReceiver (FSPL+Rx天线)
  └─ BuildCIR/PDP/APS/Statistics
```

### 3.2 SBR Coverage 模式

详见 `coverage效率优化方案.md`。核心流程：

```
SbrEngine::Run()
  └─ GenerateFibonacciRays(N)
  └─ #pragma omp parallel for schedule(guided)
       └─ while (depth<max && power>thr):
            ├─ QueryClosestFaceHitFast (BVH近远遍历)
            ├─ RxHashGrid::CheckSegment (27邻域)
            ├─ [可选] FindNearbyWedges + Keller锥绕射
            ├─ FresnelPowerReflectionCached (thread_local)
            ├─ MC透射(概率选择+物理衰减)
            └─ 反射 (确定性+物理衰减)
  └─ 归并线程结果 → 功率dBm → 导出JSON
```
