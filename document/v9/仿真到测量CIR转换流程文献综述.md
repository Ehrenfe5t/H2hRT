# 射线追踪仿真CIR到测量等效带限CIR转换流程的文献综述

## 摘要

射线追踪（Ray Tracing, RT）仿真产生的信道冲激响应（Channel Impulse Response, CIR）是理想无限带宽下的确定性信道表征，而矢量网络分析仪（VNA）或相关信道探测器实测的CIR受限于有限测量带宽、系统响应和校准程序。将理想的RT仿真CIR转换为测量等效的带限CIR，需要系统性地模拟整个测量链：从VNA校准与去嵌入、带宽限制与频谱加窗，到多径参数提取与统计验证。本文综述2020至2025年间该完整转换流程的关键文献，覆盖测量系统校准方法论、RT仿真校准策略、带宽限制的物理效应、CIR后处理技术以及仿真-测量比对验证指标，为相关方向的研究生学位论文提供系统的文献基础与实验设计参考。

**关键词：** 射线追踪；信道冲激响应；VNA校准；带限CIR；带宽限制；去嵌入；验证指标

---

## 1. 引言

射线追踪仿真与信道实测之间的"鸿沟"并非仅源于建模精度，更系统性地根植于以下差异：

| 维度 | RT仿真CIR | VNA/探测器实测CIR |
|------|----------|-------------------|
| **带宽** | 理论上无限（取决于频率采样精度） | 有限（典型值：100 MHz至4 GHz） |
| **系统响应** | 无（理想冲击响应） | 包含电缆、放大器、天线的联合传递函数 |
| **噪声与动态范围** | 无噪声（或数值噪声） | 受热噪声、相位噪声、动态范围制约 |
| **天线效应** | 方向图可按Jones矩阵精确建模 | 方向图嵌入测量CIR，需去嵌入 |
| **时延分辨率** | 受限于RT的射线可分辨性 | 受限于带宽倒数（Δτ = 1/W） |
| **频率选择性** | 连续（或极高分辨率采样） | 离散频率点采样（如801点） |

本文的目标是系统梳理从理想RT CIR到测量等效带限CIR的完整转换流程，涵盖VNA校准、RT校准、带宽限制、窗函数处理及统计验证指标五个核心环节。

## 2. 总体验证工作流

综合2020-2025年间发表的主要文献，仿真-测量比对遵循如下五阶段流水线：

```
[1] 测量活动                     [2] RT仿真设定
    │                               │
    信道探测器 (VNA/相关)          3D数字模型
    系统校准 (B2B)                 材料参数库
    频率扫描                        射线发射配置
    │                               │
    ├───────────────┬───────────────┤
                    │
          [3] 参数提取 (匹配处理)
                    │
            SAGE / RiMAX / MUSIC
            → MPC (时延, 角度, 功率)
            → PDP, PADP
                    │
          [4] 统计比对
                    │
            CDF: PL, RMS-DS, RMS-AS
            误差指标: ME, RMSE, STD
            分场景: LOS / OLOS / NLOS
                    │
          [5] 校准迭代循环
                    │
            调整材料 εr, σ, 散射系数
            修正数字地图几何误差
            迭代直至误差达标
```

该流水线的核心原则是：**仿真与测量必须在完全相同的后处理参数下进行参数提取与统计比较**——包括相同的带宽、相同的窗函数、相同的MPC提取算法及其门限值。

## 3. VNA信道探测器方法论

### 3.1 VNA信道探测的基本原理

VNA（矢量网络分析仪）信道探测器通过频率扫描测量信道的S21参数（传输系数），然后通过逆傅里叶变换获得时域CIR：

```
h(τ) = IFFT{H(f)} = IFFT{S21_calibrated(f)}
```

其中`S21_calibrated(f)`是经过系统响应去嵌入后的校准传输函数。

### 3.2 Mbugua, Fan等人 (2020) 的VNA探测器方法论

Mbugua, Fan等人发表于IET Microwaves, Antennas & Propagation (2020)的工作提供了VNA信道探测器系统开发与验证的完整方法论 [IET MAP, Vol. 14, Issue 14, pp. 1733–1741, 2020]。

**三种光纤相位补偿方案对比：**
- 方案A：单向光纤反馈——相位稳定性中等
- 方案B：带光环行器的双向方案——**最优**，在30 GHz下17小时内相位偏差<2.5°
- 方案C：双向无环行器——成本较低但隔离度受限

**CIR提取流水线：**
1. 在带宽W上测量S21(f)
2. 应用来自反馈链路的相位补偿
3. 通过背靠背（Back-to-Back, B2B）校准测量进行系统响应去嵌入
4. 逆傅里叶变换：h(τ) = F^{-1}{H(f)}
5. 20 dB射频衰减器的去嵌入（后处理）
6. SAGE算法进行多径参数估计
7. Bartlett波束赋形器计算功率角度延迟分布（PADP），使用Hanning窗

**系统性能指标：**
- 动态范围：最高112 dB
- 相位稳定性：<2.5° @ 30 GHz / 17小时
- 适用距离：长距离信道探测

### 3.3 300 GHz信道探测器的验证

IEICE Communications Express (2023, Vol. 12, No. 6) 发表了一项双向验证工作：使用RT仿真作为地面真值（ground truth）来验证300 GHz信道探测器本身的性能。这代表了验证范式的反转——不仅探测器用于验证RT，RT也可用于验证探测器的准确性，两者形成闭环验证链。

## 4. RT仿真校准工作流

### 4.1 Gougeon等人 (EuCAP 2024) ——工业毫米波校准

该工作发表于EuCAP 2024 [DOI: `10.23919/EuCAP60739.2024.10501179`, arXiv: 2404.10590]，是迄今为止最详细、最系统的RT校准方法论之一。

**测量设置：**
- 探测器类型：VNA信道探测器
- 频段：26-30 GHz（4 GHz带宽）
- 频率点数：801点（5 MHz间隔）
- 发射天线：虚拟5×5×5立方阵列（半波长间距），垂直单极子
- 接收天线：单个垂直单极子，20个位置（覆盖LOS/OLOS/NLOS）
- MPC提取：SAGE算法（停止条件：100条MPC或95%总功率）

**RT设置：**
- 引擎：Volcano Flex (SIRADEL)
- 算法：波束发射（Beam Launching），每条射线最多2次反射+2次绕射
- 3D数字模型：35×16×3.5 m机加工车间，基于平面图和照片构建
- 材料参数：ITU标准材料库
- 散射模型：金属物体上的Lambertian漫散射

**四步校准流程：**

**步骤1——几何校正：**
- 从照片中添加缺失物体（天花板管道）
- 重新定位接收点以确保NLOS遮挡对齐准确

**步骤2——射线-路径映射：**
- 按功率、时延和离开角（AoD）对预测射线与实测MPC进行匹配
- 识别缺失射线（环境中存在但RT未预测的路径）和虚假射线（RT预测但测量中不存在的路径）

**步骤3——局部衰减校正：**
- 在管道周围添加虚拟包络（20 dB/m额外损耗），用于穿过管道间隙的射线

**步骤4——全局电磁参数优化：**
- 优化反射偏移量（-3 dB）和绕射偏移量（-2 dB），以最小化所有大尺度参数（LSP）的RMSE

**最终精度：**
| 指标 | 误差 |
|------|------|
| 接收功率 | < 1.5 dB |
| RMS时延扩展 | < 5 ns |
| RMS方位角扩展 | < 2° |

### 4.2 Kanhere, Poddar & Rappaport (IEEE TAP 2024) ——NYURay校准

该工作发表于IEEE TAP 2024 [arXiv: 2410.03104]，覆盖9个测量活动（3个频率×3个环境）。

**关键创新——角度无关的反射校准方法：**
- 传统方法需要逐角度迭代优化反射系数，计算代价高
- 该工作提出了一种**角度无关的单一反射系数偏移校准**，每个环境仅需1-2次校准测量
- 在少数校准位置测量定向PDP
- 使用默认ITU材料参数运行NYURay
- 计算一个全局反射系数偏移（dB），使所有角度下校准位置的功率误差最小化
- 将该偏移全局应用，在留出位置验证

**最终精度：**
- 室内定向多径功率预测：标准差<3 dB
- 室外/工厂定向多径功率预测：标准差<2 dB
- RMS-DS和RMS-AS略被低估（因环境地图不完整）

**所用RT算法：** 混合SBR（Shooting and Bouncing Rays）+ 镜像法（Image-based）

### 4.3 Hoydis等人 (NVIDIA, IEEE TMLCN 2024) ——可微RT校准

该工作发表于IEEE TMLCN 2024 [arXiv: 2311.18558, GitHub: NVlabs/diff-rt-calibration]，代表了RT校准的范式转变。

**核心创新——将RT引擎视为可微计算图：**
- RT引擎：Sionna RT
- 材料参数、散射系数和天线方向图均作为**可训练参数**
- 梯度下降直接最小化仿真CIR与实测CIR之间的误差

**验证数据：**
- DICHASUS分布式MIMO信道探测器（斯图加特大学）
- 公开数据集：`dichasus-dc01.tfrecords`

**三种材料模型层级：**
| 模型层级 | 描述 |
|---------|------|
| ITU材料模型 | 标准表格化参数 + 可学习的功率缩放因子 |
| 可学习材料模型 | 每种表面类型的可微εr、σ，通过梯度下降优化 |
| 神经材料模型 | 位置编码的神经网络，表示连续空间变化的材料属性 |

**CIR级比对：**
- 直接比对复CIR（而非仅比对压缩的统计量）
- 损失函数作用于各MPC的路径增益、时延和角度
- 实现端到端优化，无需人工参数调节

## 5. 带宽限制的物理效应

### 5.1 基本关系

时延分辨率与测量带宽呈反比关系：

```
Δτ = 1 / W
```

| 带宽 (W) | 时延分辨率 (Δτ) | 实际含义 |
|----------|----------------|---------|
| 100 MHz | 10 ns | 路径差需>3 m才能分辨两条路径 |
| 500 MHz | 2 ns | 路径差需>0.6 m |
| 1 GHz | 1 ns | 路径差需>0.3 m |
| 2 GHz | 0.5 ns | 路径差需>0.15 m |
| 4 GHz | 0.25 ns | 路径差需>0.075 m |

### 5.2 带限CIR的数学表达

理想无限带宽CIR可表示为离散多径分量的叠加：

```
h_ideal(τ) = Σ_k α_k · δ(τ - τ_k)
```

其中α_k为复路径增益，τ_k为路径时延。

将该CIR限制在测量带宽W内，等效于在频域施加矩形窗，在时域产生sinc函数卷积：

```
h_bandlimited(τ) = h_ideal(τ) * [W · sinc(Wτ)]
                = Σ_k α_k · W · sinc(W(τ - τ_k))
```

关键的物理后果：
1. **旁瓣（Sidelobes）：** 每条路径的sinc旁瓣会干扰邻近路径，可能导致虚假路径检测或弱路径淹没。
2. **路径合并：** 当|τ_k - τ_j| < 1/W时，两条路径模糊为单个展宽的峰值。
3. **功率泄露：** 路径功率不再集中在单一采样点，而是在时延域展宽。

### 5.3 窗函数的选择

为抑制sinc旁瓣，在逆傅里叶变换前对频域数据施加窗函数：

| 窗函数 | 主瓣宽度 | 最高旁瓣 | 适用场景 |
|--------|---------|---------|---------|
| 矩形窗 | 1.00× | -13 dB | 最高分辨率，适用于稀疏多径 |
| Hanning窗 | 2.00× | -32 dB | 平衡分辨率与旁瓣抑制，**最常用** |
| Hamming窗 | 1.85× | -43 dB | 更强的旁瓣抑制 |
| Blackman窗 | 3.00× | -58 dB | 密集多径场景 |
| Kaiser窗 (β=6) | 2.76× | -44 dB | 可调参数，灵活折中 |

**处理建议：** RT仿真CIR和实测CIR必须在完全相同的窗函数和带宽参数下进行处理，否则比对结果将因窗函数差异而引入系统性偏差。所有文献一致推荐使用**Hanning窗**作为标准选择 [Mbugua, Fan et al., 2020; Gougeon et al., 2024]。

## 6. 仿真CIR到测量等效CIR的完整转换程序

基于上述文献的综合，在RT仿真CIR与实测CIR之间建立可比性，需执行以下转换步骤：

### 步骤1：RT CIR的生成（理想环境）

从RT引擎获取理想CIR：h_rt(τ) = Σ_k α_k^rt · δ(τ - τ_k^rt)，其中每条射线的α_k包含完整的Jones极化矩阵（2×2）。

### 步骤2：系统响应卷积（模拟VNA传递函数）

测量系统的传递函数h_sys(τ)（通过B2B校准测量获得）与理想RT CIR进行卷积：

```
h_rt_sys(τ) = h_rt(τ) * h_sys(τ)
```

或等效地在频域相乘：H_rt_sys(f) = H_rt(f) · H_sys(f)。

若缺乏实测系统响应数据，可近似为理想带通滤波器：
```
H_sys(f) = rect((f - f_c) / W)
```
其中f_c为中心频率，W为带宽。

### 步骤3：带宽限制

仅保留[H_sys(f)]非零频带内的频谱分量，或等效地：
```
H_limited(f) = H_rt(f) · H_sys(f)
h_limited(τ) = IFFT{H_limited(f)}
```

### 步骤4：频谱加窗

对频域数据施加窗函数w(f)（如Hanning窗）：
```
H_windowed(f) = H_limited(f) · w(f)
h_windowed(τ) = IFFT{H_windowed(f)}
```

### 步骤5：频率采样离散化

将连续频谱在测量频点上采样（如801点）：
```
H_disc[k] = H_windowed(f_k),  k = 0, 1, ..., N-1
```

### 步骤6：添加噪声（可选）

根据实测系统的噪声基底，向离散频域样本添加复高斯白噪声，以匹配实测的动态范围。

### 步骤7：MPC参数提取

使用与实测数据**完全相同**的MPC提取算法（SAGE/RiMAX/MUSIC）和参数设置，从带限CIR中提取多径参数（时延、角度、功率）。

## 7. 验证指标与统计比对方法

### 7.1 大尺度参数（LSP）

标准的验证指标集包括：

| 参数 | 符号 | 单位 | 典型误差目标 |
|------|------|------|-------------|
| 路径损耗 | PL | dB | ME < 1 dB, RMSE < 2 dB |
| RMS时延扩展 | σ_τ (DS) | ns 或 log10(s) | RMSE < 5 ns [Gougeon 2024] |
| RMS方位角扩展 | σ_φ (AS) | deg 或 log10(deg) | RMSE < 2-5 deg |
| RMS仰角扩展 | σ_θ (ES) | deg | 依场景而定 |
| Rician K因子 | K | dB | 依场景而定 |

### 7.2 误差指标

- **平均误差（ME）：** ME = (1/N) Σ (x_sim - x_meas) ——衡量系统性偏差
- **均方根误差（RMSE）：** RMSE = sqrt[(1/N) Σ (x_sim - x_meas)^2] ——衡量整体偏离
- **标准差（STD）：** STD = sqrt[(1/N) Σ (x_sim - x_meas - ME)^2] ——衡量随机波动

### 7.3 统计比对方法

- **累积分布函数（CDF）比对：** 绘制仿真与测量的LSP CDF曲线，直观展示分布一致性
- **散点图：** 每个测量位置的仿真值 vs 实测值，理想情况下点应聚集在y=x对角线附近
- **分场景拆解：** 分别报告LOS、OLOS和NLOS条件下的误差，因为不同传播条件的建模难度差异显著

### 7.4 MPC级比对（高阶验证）

Hoydis等人 (2024) 倡导的CIR级比对：

- **路径匹配：** 在时延-角度域将仿真射线与实测MPC进行配对
- **逐路径功率误差：** 评估单个主导路径的功率预测精度
- **缺失路径率：** 实测中存在而仿真未预测的路径占比
- **虚假路径率：** 仿真预测而实测不存在的路径占比

## 9. 总结与实验设计建议

### 9.1 核心原则

1. **处理一致性：** 仿真和测量CIR必须在相同的带宽、窗函数、MPC提取算法和门限下处理。
2. **系统响应不可忽略：** RT仿真CIR在带宽限制前必须先与系统响应卷积（或频域相乘），否则路径功率与时延的对应关系将产生系统性偏差。
3. **校准是迭代过程：** 从初始RT仿真到最终验证通过的仿真，通常需要3-5轮迭代校准（几何校正→局部衰减→全局参数→再验证）。
4. **分场景报告：** 不同传播条件（LOS/OLOS/NLOS）的误差水平差异显著，必须分场景提供独立的误差统计。

### 9.2 推荐实验流水线

1. **测量活动：** 使用VNA或相关探测器，在目标频段以目标带宽进行频率扫描，记录B2B校准数据。
2. **初始RT仿真：** 基于数字地图和ITU材料库运行RT，输出理想CIR（全Jones极化矩阵）。
3. **转换处理：** 按第6节的六步程序将RT CIR转换为测量等效带限CIR。
4. **参数提取：** 使用SAGE算法对仿真和测量CIR分别提取MPC参数和LSP。
5. **统计比对：** 计算PL、DS、AS的ME/RMSE/STD，绘制CDF比对图和散点图。
6. **校准迭代：** 若误差未达标，按Gougeon等人的四步法进行校准（几何→射线匹配→局部衰减→全局参数），重复步骤3-5。
7. **最终验证：** 在留出位置（未参与校准的测量点）上报告最终误差指标。

### 9.3 面向Jones矢量RT仿真平台的扩展建议

对于采用全极化Jones矢量的RT仿真平台，上述校准流程需额外关注：

- **极化分集验证：** 不仅验证共极化链路的LSP，还需独立验证交叉极化链路的路径损耗和XPD。
- **极化窗函数：** 若测量使用了极化切换，仿真CIR需模拟相同的极化切换时序。
- **天线去嵌入：** 若测量天线方向图已知，应在参数提取前对实测数据进行去嵌入 [Li et al., 2024, DOI: `10.1109/TAP.2023.3324030`]，或在仿真端施加相同的天线方向图。
