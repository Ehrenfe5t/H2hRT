### 3.3 多属性天线耦合建模

| 耦合类型 | 当前代码支持 | 缺失情况 |
|---------|:---:|------|
| 方向图+极化联合 | ✅ `InitializeTxField` 同时加载增益+极化 | 已就位 |
| 姿态×极化联合 | ✅ `WorldToAntennaSpherical` + Jones pattern | 已就位 |
| Rx 共轭极化匹配 | ✅ `vr = E_inc · conj(h_rx)` | 已就位 |
| 天线互耦合 | ❌ 无（S参数矩阵） | **超出硕士范围** |
| 天线-人体耦合 | ❌ 无 | **超出硕士范围** |
| 阵列波束赋形 | ❌ 无 | **超出硕士范围** |

**论文可写内容**：方向图×极化×姿态三者联合——这是论文的核心"多属性耦合"。当前代码已实现。互耦合可标注为"假设天线间距足够大，忽略互耦合效应"。

**仿真需求**：
1. 同一发射天线，不同 Rx 天线极化 → 验证交叉极化隔离
2. 同一天线，不同姿态旋转 → 验证姿态对接收功率的影响
3. 方向图+极化+姿态三属性同时变化 → 验证耦合效应

## 五、论文第三章写作建议

### 3.1 节核心论述结构

```
3.1.1 天线方向图属性 → 标量增益 dBi vs 复 Jones 方向图
3.1.2 天线极化属性 → 线极化/圆极化/椭圆极化，Ludwig-3 定义
3.1.3 天线姿态属性 → forward/up 定义，Euler 角 vs 四元数
3.1.4 天线频率属性 → 频率响应/群时延/阻抗（标注待扩展）
```

### 3.2 节核心论述结构

```
3.2.1 天线方向图数据格式 → theta/phi/gain CSV
3.2.2 Jones 极化数据格式 → theta/phi/Etheta_re/im/Ephi_re/im CSV
3.2.3 天线数据库构建 → 按类型/频率索引（当前支持 3 种示例天线）
3.2.4 方向图查询与插值 → 角度双线性插值（已实现）
```

### 3.3 节核心论述结构

```
3.3.1 方向图×极化耦合 → Jones 方向图 = E_theta × theta_hat + E_phi × phi_hat
3.3.2 姿态×极化耦合 → 旋转后的本地坐标 → 世界坐标变换
3.3.3 三属性联合模型 → E_world(tx) = R(pose) × [E_theta, E_phi]
3.3.4 Rx 端共轭匹配 → vr = E_inc · conj(h_rx)（已实现）
```

### 3.4 节核心论述结构

```
3.4.1 Tx 天线注入 RT 链路 → InitializeTxField
3.4.2 沿路径复矢量传播 → ApplyFreeSpace + ApplyReflection/Transmission/Diffraction
3.4.3 Rx 天线接收 → FinalizeAtReceiver（共轭匹配）
3.4.4 天线对信道特性的影响 → PDP/APS/K因子/覆盖热图
```
