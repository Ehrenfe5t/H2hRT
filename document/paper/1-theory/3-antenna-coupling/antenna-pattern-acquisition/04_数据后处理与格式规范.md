# 04 — 数据后处理与格式规范

> 目的：将测量/仿真原始数据转换为 H2hRT 可读的标准 CSV 格式  
> 输入：VNA S21 数据 / HFSS 远场导出数据 / 理论公式计算结果  
> 输出：`theta_deg, phi_deg, gain_dBi, Etheta_re, Etheta_im, Ephi_re, Ephi_im`

---

## 4.1 原始数据到方向图的转换链

```
实测：VNA S21(dB) → Friis 公式 → Gain(θ,φ) → 坐标对齐 → CSV
仿真：HFSS 远场 R → Gain(θ,φ) + Eθ/Eφ 复值 → 坐标对齐 → CSV
理论：解析公式 → Gain(θ,φ) + 极化 → CSV
```

---

## 4.2 实测数据：S21 → 增益

### 4.2.1 Friis 传输公式（远场形式）

$$
\frac{P_r}{P_t} = G_t(\theta_t, \phi_t) \cdot G_r(\theta_r, \phi_r) \cdot \left(\frac{\lambda}{4\pi d}\right)^2
$$

在 dB 域：

$$
|S_{21}|_{\text{dB}} = G_{t,\text{dBi}} + G_{r,\text{dBi}} + 20\log_{10}\left(\frac{\lambda}{4\pi d}\right)
$$

### 4.2.2 增益对比法（Gain Comparison, IEEE Std 149 §12.3）

用已知增益的参考天线（G_ref）标定 AUT：

$$
G_{\text{AUT}}(\theta, \phi) = G_{\text{ref}} + (S_{21,\text{AUT}}(\theta, \phi) - S_{21,\text{ref}})
$$

**步骤**：
1. Tx = 参考天线，Rx = AUT → 测量 S21_AUT(θ,φ)
2. Tx = 参考天线，Rx = 相同型号的参考天线 → 测量 S21_ref（仅 boresight）
3. 方向图：G_AUT(θ,φ) = G_ref_known + ΔS21(θ,φ)

### 4.2.3 增益对比法的可选形式

| 方法 | 天线数量 | 适用场景 |
|---|---|---|
| 两天线法 | 2（已知增益 + AUT） | 有标准增益天线 |
| 三天线法 | 3（全未知增益） | 无标准增益天线——通过三组配对测量联立求解 |
| NIST 增强增益外推法 | 2 + 精密定位 | 计量级精度 ±0.07 dB (Gordon & Moser, 2024) |

### 4.2.4 近场修正

当测量距离 $d < 2D^2/\lambda$ 时：

$$
G_{\text{far-field}} = G_{\text{measured}} + \Delta_{\text{near-field}}
$$

近场修正 Δ 取决于天线类型和 d/D²/λ：
- d 在 1.0–2.0 × (2D²/λ)：Δ ≈ 0.3–1.0 dB
- d 在 0.5–1.0 × (2D²/λ)：Δ ≈ 1.0–3.0 dB（不建议）
- d < 0.5 × (2D²/λ)：Δ 不可靠，必须用近场扫描法

**强烈建议**：确保 d ≥ 2D²/λ。

## 4.3 坐标系统对齐

### 4.3.1 测量坐标系 → 天线局部坐标系

测量时 Tx 和 Rx 的 boresight 对准方向定义为天线局部坐标系：
- **Z 轴（theta=0）** = boresight 方向（天线主瓣指向）
- **X 轴（phi=0）** = 水平面内垂直于 Z 的方向（通常为 E 面或 H 面）
- **Y 轴** = 右手定则确定（X × Y = Z）

### 4.3.2 实测 S21 角度映射

实测时：

| 实测动作 | 对应的 (θ, φ) |
|---|---|
| Rx 绕垂直轴旋转（方位） | φ 变化（固定 θ=90° 为 H 面扫描） |
| Rx 绕水平轴旋转（俯仰） | θ 变化（固定 φ=0° 为 E 面扫描） |

**完整球面覆盖**：旋转 AUT 的 φ（0°–360°），对于每个 φ，改变 θ（0°–180°）。实际中通常测两个正交面（E 面 + H 面），主截面外的方向用插值或仿真补充。

---

## 4.4 极化提取

### 4.4.1 实测极化

实测极化需要：
1. **两次正交极化测量**：Tx 和 Rx 分别用垂直极化和水平极化天线，共 4 组测量（VV、VH、HV、HH）
2. 从 4 组 S21 数据中提取复极化矩阵：
   $$
   \mathbf{J} = \begin{bmatrix} E_{\theta,\text{co}} & E_{\theta,\text{cross}} \\ E_{\phi,\text{cross}} & E_{\phi,\text{co}} \end{bmatrix}
   $$
3. 归一化后得到 Ludwig-3 的 polTheta 和 polPhi 复值

### 4.4.2 仿真极化

HFSS 直接提供 Eθ 和 Eφ 的复分量（rETheta Real/Imag, rEPhi Real/Imag），可直接用于 CSV。

## 4.5 CSV 格式规范

### 4.5.1 H2hRT 标准格式

```
theta_deg,phi_deg,gain_dBi,Etheta_re,Etheta_im,Ephi_re,Ephi_im
```

**列定义**：

| 列 | 类型 | 范围 | 说明 |
|---|---|---|---|
| theta_deg | float | [0, 180] | 仰角（0° = boresight），步长 5° |
| phi_deg | float | [0, 360] | 方位角（0° = X 轴 / E 面 起始），步长 5° |
| gain_dBi | float | [−40, +20] | 增益 (dBi)，包含天线效率但不含阻抗失配 |
| Etheta_re | float | [−1, 1] | Ludwig-3 共极化分量 |Eθ| 的实部（归一化后） |
| Etheta_im | float | [−1, 1] | 虚部 |
| Ephi_re | float | [−1, 1] | 交叉极化分量 |Eφ| 的实部（归一化后） |
| Ephi_im | float | [−1, 1] | 虚部 |

**归一化约定**：极化分量归一化使得 $|E_\theta|^2 + |E_\phi|^2 = 1$（每个采样点功率归一化）。

### 4.5.2 网格要求

- **矩形网格**：每个 θ 面的 φ 值必须完全一致（θ × φ 矩形采样的定义）
- **φ 覆盖**：[0, 360) → φ=360 允许但 φ=0 等效
- **θ 覆盖**：[0, 180] → 包含 boresight 和背瓣
- **周期边界**：φ=0 和 φ=360 的增益值必须是连续的

### 4.5.3 验证检查

```python
# 格式验证脚本
import numpy as np
data = np.loadtxt('antenna_pattern.csv', delimiter=',', skiprows=1)
theta = np.unique(data[:,0])
phi = np.unique(data[:,1])

assert n_theta * n_phi == len(data), f"非矩形网格: {n_theta}×{n_phi}≠{len(data)}"
assert phi[-1] - phi[0] > 300, f"φ 覆盖不足: {phi[-1]-phi[0]}°"
print(f"网格有效: {n_theta} θ × {n_phi} φ")
```

## 4.6 实测数据后处理脚本模板

```python
"""
s21_to_gain_csv.py
将 VNA 测量的 S21 幅度/相位数据转换为 H2hRT 增益 CSV。
输入：S21 数据文件（freq, theta, phi, |S21|_dB, phase_deg）
输出：标准 CSV
"""
import numpy as np

def s21_to_gain(s21_db, g_ref_dbi, s21_ref_db):
    """增益对比法：AUT 增益 = G_ref + (S21_AUT - S21_ref)"""
    return g_ref_dbi + (s21_db - s21_ref_db)

def gain_to_field_components(gain_dbi, pol='vertical'):
    """增益转归一化电场分量（垂直极化）"""
    gain_lin = 10 ** (gain_dbi / 10)
    amp = np.sqrt(gain_lin)
    if pol == 'vertical':
        # Co-pol = E_theta (沿 theta_hat), Cross-pol = E_phi
        et_re = amp  # 归一化假设
        et_im = 0.0
        ep_re = 0.0  # 理想纯极化，无交叉分量
        ep_im = 0.0
    return et_re, et_im, ep_re, ep_im
```
