# 00 — 概述与路线图

> 文档体系：天线方向图与极化方向图获取全链路  
> 定位：论文第三章"天线精细化建模对信道指标的可量化影响"的数据基础  
> 版本：v1.0 / 2026-06-29

---

## 0.1 文档目的

本系列文档为 H2hRT v11 论文实验提供**真实天线方向图与极化方向图**的获取全链路方法论，覆盖三种数据来源：

| 来源 | 获取方式 | 适用天线 | 精度 |
|---|---|---|---|
| **理论解析解** | 闭式公式 / 孔径场法 | 偶极子 ✓ / 双锥 △ / 喇叭 ✓ | 解析精度，有限结构需修正 |
| **全波仿真** | HFSS / CST FEM 求解 | 三种天线均可 ✓ | 取决于网格收敛和材料参数 |
| **实测** | 暗室 / VNA / 远场/近场扫描 | 仅限有实物天线（双锥 + 喇叭） | 取决于设备和环境（±0.3 ~ ±1.5 dB） |

最终产出：每种天线 × 每种来源 → `theta_deg, phi_deg, gain_dBi, Etheta_re, Etheta_im, Ephi_re, Ephi_im` CSV → 导入 H2hRT。

## 0.2 三维数据矩阵

| 天线 | 理论来源 | 全波仿真 | 实测 |
|---|---|---|---|
| **半波偶极子** (2.4 GHz) | ✅ 解析解 `G(θ)=cos²(π/2·cosθ)/sin²θ` | ✅ HFSS 可仿真 | ❌ 无实物（参考基准） |
| **双锥天线** SZ-2001800/P (2–18 GHz) | △ Schelkunoff 无限长解，有限长需修正 | ✅ HFSS 全波仿真（含端部效应） | ✅ VNA + 暗室实测 |
| **标准增益喇叭** LB-75-10-C-NF (10–15 GHz) | ✅ 孔径场法（H面/E面分列） | ✅ HFSS 全波仿真 | ✅ VNA + 暗室实测 |

符号：✅ 可直接获取 / △ 近似可用，精度受限 / ❌ 不可用

## 0.3 交叉对比框架

```
                    理论解析解 ──────────┐
                       │                 │
                       ▼                 ▼
                   全波仿真 ◀──────▶ 实测数据
                       │                 │
                       └────────┬────────┘
                                ▼
                      导入 H2hRT RT 引擎
                                │
                                ▼
                      PDP / APS / XPR / K / RMS
                                │
                                ▼
            ┌───────────────────┼───────────────────┐
            ▼                   ▼                   ▼
      理论 vs 仿真          仿真 vs 实测         理论 vs 实测
      (验证仿真精度)       (验证实测可信度)     (验证理论模型)
      
      论文核心贡献：
      "天线建模精度对信道指标的影响量化"
```

**三个闭合对比环**：

| 对比环 | 问题 | 预期结论 |
|---|---|---|
| 理论 vs 仿真 | 全波仿真与解析解的差异是否 < 1 dB？ | 差异 < 0.5 dB → 仿真可替代理论（无解析解的天线） |
| 仿真 vs 实测 | 实测方向图与仿真是否一致？偏差源是什么？ | 暗室精度 ±0.5 dB 内一致 → 仿真可指导实测 |
| 导入 RT 后对比 | 不同天线数据源的 RT 结果差异能否追溯到具体的方向图差异？ | 建立"方向图差异 → 信道指标差异"的定量溯源链 |

## 0.4 天线实物简介

### SZ-2001800/P — 宽带双锥天线

| 参数 | 值 |
|---|---|
| 制造商/型号 | ETS-Lindgren / SZ-2001800/P（或同类国产替代） |
| 频率范围 | 2 – 18 GHz |
| 标称增益 | 0 – 4 dBi（频段中段接近 isotropic） |
| 极化 | 垂直极化（双锥轴沿竖直方向） |
| VSWR | ≤ 2.5:1（典型 ≤ 2.0:1） |
| 方向图特征 | 水平面全向（±1 dB 内），仰角面类似短偶极子但更宽 |
| 用途 | 超宽带全向接收 / 发射，EMC 测试标准天线 |

### LB-75-10-C-NF — 标准增益喇叭

| 参数 | 值 |
|---|---|
| 制造商/型号 | A-INFO / LB-75-10-C-NF（或同类） |
| 频率范围 | 10 – 15 GHz（WR-75 波导频段） |
| 标称增益 | 15 dBi（频段中心，±1 dB） |
| 极化 | 线性极化（沿波导宽边） |
| VSWR | ≤ 1.5:1 |
| 3 dB 波束宽度 | E面 ~20°，H面 ~25° |
| 用途 | 增益标准、定向链路、雷达截面测量参考 |
