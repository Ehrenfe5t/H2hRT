# 03 — 天线方向图实测方案

> 目的：为 SZ-2001800/P（双锥）和 LB-75-10-C-NF（喇叭）建立三级实测方案  
> 方案分级：经济型 / 标准型 / 精密型，逐级提升精度和预算  
> 关键参考：IEEE Std 149-2021, Balanis Ch.17, ITU-R SM.2056

---

## 3.0 三种方案总览

| | 方案 A：经济型 | 方案 B：标准型 | 方案 C：精密型 |
|---|---|---|---|
| **预算** | $5k–15k（已有 VNA 可再降） | $50k–150k | $300k+ |
| **精度** (增益) | ±1.0–1.5 dB | ±0.5–0.8 dB | ±0.2–0.4 dB |
| **精度** (方向图) | ±2–3 dB（旁瓣） | ±1–2 dB | ±0.5 dB |
| **场地** | 普通实验室 / 开放空间 | 小型屏蔽暗室 | 全电波暗室 / 近场扫描暗室 |
| **核心设备** | VNA + 便携吸波材料 + 手动转台 | 紧凑暗室 + 自动定位器 + VNA | 全电波暗室 + 球面近场扫描仪 + 高性能 VNA |

| **适合场景** | 初步验证 / 论文预实验 | 正式论文数据 / 增益标定 | 计量级精度 / 天线认证 |

## 3.1 方案 A：经济型（Budget Tier）

|---|---|---|---|---|

**总预算**：~$30k（若有 VNA 则 ~$5k）

### 3.1.2 测量几何

```
                    d (远场距离)
        Tx ─────────────────────────── Rx
        │                              │
        ▼                              ▼
    发射天线                          待测天线 (AUT)
   (标准增益天线)                    (双锥或喇叭)
        │                              │
        └──────── VNA ────────────────┘
                Port 1     Port 2
```

**远场条件**：$d \geq \frac{2D^2}{\lambda}$，其中 D = 天线最大孔径尺寸

| 天线 | 频率 | D | λ | 最小远场距离 |
|---|---|---|---|---|
| 喇叭 (15 GHz) | 15 GHz | ~0.12 m | 0.02 m | 1.44 m |
| 双锥 (2 GHz) | 2 GHz | ~0.2 m | 0.15 m | 0.53 m |
| 双锥 (18 GHz) | 18 GHz | ~0.2 m | 0.017 m | 4.7 m |

**推荐距离**：**5 m**（覆盖双锥 18 GHz 的远场条件，并有一定余量）

### 3.1.3 操作流程

**步骤 1：VNA 校准**
- 执行完整的 2 端口 SOLT 校准（Short-Open-Load-Through）
- 校准面定在电缆末端（天线连接器处）
- 若电缆长 > 5 m，建议使用 VNA 的端口扩展或去嵌入功能

**步骤 2：吸波材料布置**
- 地板：Tx-Rx 连线正下方的地面上铺设 3–5 块吸波材料
- 侧面：最近的墙壁方向放置 2–3 块吸波材料
- 天花板：若高度 < 3 m，悬挂吸波材料（或用高吸波地毯）
- **关键**：Tx 和 Rx 天线背后的反射（支架、操作人员）需用吸波材料遮挡

**步骤 3：天线对准**
- 激光从 Tx 天线中心指向 Rx 天线中心
- 确保两天线同高度、boresight 对准
- 用水平仪确认天线支架垂直

**步骤 4：S21 传输测量**
- VNA 设置：频率范围覆盖待测天线频段，IF 带宽 = 100 Hz（降低噪声底）
- 点数 ≥ 1001（频率步长满足远场条件的变化）
- 平均次数 ≥ 8（降低随机噪声）
- 记录 S21(f) 复数值（幅度 + 相位）

**步骤 5：方向图扫描**
- 旋转 Rx 天线支架（手动旋台，步长 5°–10°）
- 每个 φ 角度记录 S21 值
- 重复 E 面（φ=0°–360° 在一个仰角面）和 H 面（改变仰角）

**步骤 6：增益标定**
- 用**增益对比法**（Gain Comparison Method, IEEE Std 149 §12.3）：
  - 先测量参考天线（已知增益 G_ref）的 S21_ref
  - 再测量 AUT 的 S21_AUT
  - $G_{\text{AUT}}(\text{dB}) = G_{\text{ref}}(\text{dB}) + (S_{21,\text{AUT}} - S_{21,\text{ref}})(\text{dB})$
- 参考天线：半波偶极子（2.15 dBi @ 2.4 GHz），或标准增益喇叭

### 3.1.4 主要误差源

| 误差源 | 量级 | 缓解措施 |
|---|---|---|
| 地面/墙壁多径反射 | ±1–3 dB（频率和位置依赖） | 增加吸波材料覆盖面积；使用时间门控（VNA gating）过滤长延迟反射 |
| 天线对准偏差 | ±0.5–1 dB | 激光对准 + 俯仰/方位反复微调至 S21 最大 |
| 近场效应（d < 2D²/λ） | ±1–2 dB | 严格确保 d ≥ 远场条件；若不满足，用近场修正公式 |
| 电缆弯曲/移动 | ±0.3–0.5 dB（相位敏感） | 旋转 AUT 时确保电缆不移动；使用电缆去应力夹具 |
| 阻抗失配 | ±0.2–0.9 dB | 外接 6 dB 衰减器（pad）可降低不确定度至 ±0.24 dB（Gordon & Moser, 2024） |
| VNA 噪声底 | ±0.1 dB（信号充足时） | 降低 IF 带宽、增加平均次数、使用 LNA |

### 3.1.5 精度评估

增益总不确定度（RSS 合成）：

$$
U_G \approx \sqrt{1.5^2 + 0.7^2 + 0.5^2 + 0.3^2 + 0.3^2} \approx \pm 1.8 \text{ dB}
$$

**结论**：方案 A 适合方向图主瓣和 −10 dB 内的旁瓣测量。不适合高精度增益绝对标定。

## 3.2 方案 B：标准型（Standard Tier）

### 3.2.1 硬件清单

| 设备 | 推荐型号 | 单价 (USD) |
|---|---|---|
| VNA | Keysight E5080B (9 kHz–20 GHz) | ~$60k |
| 紧凑屏蔽暗室 | ETS-Lindgren SpaceSaver 系列 (~3m×2m×2m) | ~$40k |
| 自动精密转台 | NSI-MI / Orbit FR 多轴定位器 | ~$20k |
| 标准增益喇叭（基准） | A-INFO LB-75-10（15 dBi @ 12.5 GHz） | ~$1.5k |
| LNA + 电缆 + 校准件 | — | ~$5k |

**总预算**：~$130k

### 3.2.2 关键升级点

| 方案 A 限制 | 方案 B 改进 | 精度提升 |
|---|---|---|
| 地面反射需手动贴吸波 | 全屏蔽暗室（−40 dB 反射率） | −1.0 dB 误差 |
| 手动旋台步长粗糙 | 自动转台 ±0.1° 重复精度 | −0.5 dB 误差 |
| VNA 噪声底限制 | 高性能 VNA + LNA | −0.2 dB 噪声 |
| 人工对准不一致 | 激光 + 自动对准程序 | −0.3 dB 误差 |

### 3.2.3 操作流程

与方案 A 基本相同，但：
1. **自动扫描**：转台软件控制步进（1° 步长，θ=0–360°, φ=0–180°）
2. **VNA 编程控制**：通过 SCPI/GPIB 自动采集 S21
3. **时间门控**：利用 VNA 的 Time Domain Gating 滤除残余多径（门宽 ~2 ns，等效 60 cm 空间分辨率）
4. **阻抗失配修正**：用 6 dB pad 方法降低失配不确定度

### 3.2.4 精度评估

$$
U_G \approx \sqrt{0.5^2 + 0.3^2 + 0.2^2 + 0.2^2 + 0.2^2} \approx \pm 0.7 \text{ dB}
$$

## 3.3 方案 C：精密型（Precision Tier）

### 3.3.1 硬件清单

| 设备 | 推荐型号 | 单价 (USD) |
|---|---|---|
| 全电波暗室 | ETS-Lindgren FACT 系列（>7m 静区） | ~$300k+ |
| 球面近场扫描仪 | NSI-MI 700S-360 系列 | ~$200k |
| 高性能 VNA | Keysight PNA-X N5247B (10 MHz–67 GHz) | ~$200k |
| 精密标准增益天线 | NIST 校准过的增益标准 | ~$5k |

**总预算**：~$700k+（通常为国家级/企业级设施）

### 3.3.2 方法：球面近场扫描

不同于远场法（方案 A/B），方案 C 使用**球面近场扫描 + 近远场变换（NFFFT）**：

1. AUT 固定，探针在包围 AUT 的球面上扫描近场幅度和相位
2. 球面波展开（SWE）将近场变换为远场方向图
3. 优势：不受远场距离限制、可测全空间方向图（包括背瓣）、无需移动 AUT

### 3.3.3 精度评估

增益不确定度：**±0.2–0.3 dB**（符合 NIST 可溯源标准）  
方向图不确定度：**±0.5 dB**（主瓣和 −30 dB 旁瓣）

## 3.4 方案选择决策矩阵

| 方案 | 预算 | 精度 | 论文可用性 | 推荐场景 |
|---|---|---|---|---|
| A 经济型 | <$10k | ±1.5 dB | 预实验 / 趋势验证 | 课题组内部验证、方法排除 |
| B 标准型 | ~$130k | ±0.7 dB | **正式论文数据** | 若课题组有暗室或可借用 |
| C 精密型 | >$700k | ±0.3 dB | 计量级验证 | 若可借用国家级/企业级设施 |

**推荐路径**：若课题组无暗室 → 方案 A 做预实验 + 借用合作单位方案 B 设施做正式数据。

## 3.5 实施检查清单

- [ ] 确认 VNA 频率上限 ≥ 18 GHz（双锥最高频）
- [ ] 确认 VNA 已用 SOLT 校准件校准（记录校准日期和有效期）
- [ ] 远场距离验证：$d \geq 2D^2/\lambda$ 对所有频点
- [ ] 吸波材料反射率 < −30 dB @ 工作频段
- [ ] Tx 和 Rx 天线 boresight 对准（激光验证）
- [ ] 极化对准：参考天线和 AUT 的极化方向一致
- [ ] 环境干扰排除：关闭附近电子设备、WiFi 路由器、手机
- [ ] 数据记录：每个角度/频率记录原始 S21（复数值，不做任何平均/平滑）
- [ ] 增益标定：至少用一个已知增益标准做增益对比
- [ ] 重复性验证：选择 3 个角度重复测量，S21 差异 < 0.3 dB
