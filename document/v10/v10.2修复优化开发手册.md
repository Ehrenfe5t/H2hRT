## 0. 第三章研究方向再确认

### 0.1 研究方向一的本质

第三章的核心创新不是"发明一种新的射线追踪算法"，而是证明一个被现有RT工具普遍忽视的事实：

> **传统RT中"标量增益 + 固定极化矢量"的天线简化模型，在方向图、极化状态和姿态三个维度上与精细化天线模型（逐角度增益方向图 + Ludwig-3 Jones复极化 + 三维姿态）存在系统性差异，这些差异会导致 PDP、APS、XPR、MEG 等可观测信道指标出现不可忽略的偏差。**

换言之：
- **自变量**：天线模型精细度（Ideal → Dipole → Patch → Horn）和天线姿态（yaw/pitch/roll）
- **因变量**：CIR、PDP、APS、XPR分布、总接收功率、RMS delay spread、MEG
- **控制变量**：场景几何、Tx/Rx位置、SBR寻径参数（ray_count等）
- **不变前提**：RT引擎生成的天线无关几何路径集合（传播信道）在对比实验中相同

### 0.2 与文献的理论对齐

| 理论支撑 | 文献 | 在第三章中的用法 |
|---------|------|----------------|
| 双方向信道：传播信道与天线响应可分离 | Steinbauer et al. 2001 | 固定几何路径集，仅切换天线模型，观察信道指标变化 |
| MEG = 天线方向图 × 入射APS的角域内积 | Taga 1990; Kalliola 2002 | 用仿真APS和天线方向图计算MEG-like指标，解释总接收功率差异 |
| Jones矢量完整描述天线极化选择性 | Jones 1941; Ludwig 1973 | 逐角度复极化查询 → TE/TM分解 → Rx共轭匹配 → co/cross-pol功率 |
| 极化RT到CIR的全链路 | Rasp et al. 2024 | 路径级复振幅保留、反射/透射/绕射的极化演化、接收端Jones响应 |
| XPR/XPD是传播和天线的联合结果 | Shafi et al. 2006; 3GPP TR 38.901 | 对比固定极化vs逐角度Jones极化对XPR分布的影响 |

### 0.3 论文可主张的创新表述

1. 提出一种与几何RT引擎**解耦**的精细化天线响应模型——在 Ludwig-3 框架下统一方向图、Jones极化和三维姿态，作为EM链路的可替换前端/后端模块。
2. 在自研零依赖RT引擎中实现**完整复矢量EM链路**：Tx Jones注入 → 传播路径TE/TM极化演化 → Rx共轭匹配 → 路径级co/cross-pol分解。
3. 通过**V10统一增强型SBR**生成具备反射/透射/绕射多样交互机制的几何路径集合，以残差过滤和收敛性实验保证路径可用性。
4. 系统比较精细化天线模型与简化天线模型在PDP、APS、XPR、MEG等**可观测信道指标**上的差异，量化传统简化假设引入的误差。

### 1.2 每个实验需要输出的数据

#### 路径级（per-path，每条几何路径一行）
```
- path_id, delay_s, total_length_m
- amplitude_real, amplitude_imag (复振幅)
- power_linear, power_dBm
- aod_theta_deg, aod_phi_deg, aoa_theta_deg, aoa_phi_deg
- interaction_sequence (e.g., "Tx-R-Rx", "Tx-T-Rx", "Tx-R-D-Rx")
- co_pol_power_linear, cross_pol_power_linear, xpr_dB
- geometry_residual
- tx_antenna_id, rx_antenna_id, antenna_source_type
```

#### 信道级（per-Tx-Rx-pair）
```
- CIR: complex taps (delay, amplitude_real, amplitude_imag)
- PDP: power taps (delay, coherent_power, incoherent_power)
- APS: 2D theta-phi grid (power or normalized power per angular bin)
- XPR statistics: mean, median, P10/P90, power-weighted mean
- MEG-like: effective gain computed as ∫G(θ,φ)×APS(θ,φ) dΩ
- Channel statistics: total received power (dBm), RMS delay spread (s),
  K-factor (dB), effective path count, strongest path power (dBm)
```

#### 诊断级（per-run）
```
- ray_count, active_ray_peak, generated_reflection/transmission/diffraction_branches
- pruned_by_power/depth/rx_reachability, wedge_candidates_queried/coupled/rejected
- paths_before_dedup, paths_after_dedup, paths_after_postprocess
- geometry_residual_max, geometry_residual_mean
- runtime_s, memory_peak_mb
```

#### P1-2: MEG-like 指标计算

**问题**：论文3.1.5节引用了Taga的MEG公式，但未在代码中实现。需要基于仿真APS和天线方向图计算MEG-like指标。

**数学基础**（Taga 1990, Kalliola 2002）：
```
MEG = ∫₀²π ∫₀^π G(θ,φ) × P_Ω(θ,φ) × sin(θ) dθ dφ
```
其中 `G(θ,φ)` 为天线增益方向图（线性），`P_Ω(θ,φ)` 为入射波归一化角功率谱。

**修改范围**：
1. `core/em/BuildChannelStatistics.cpp`：新增 `ComputeMEG` 函数
   - 输入：`APSResult` (2D grid) + `AntennaModel` (gain pattern)
   - 对每个(θ,φ) bin：`power_lin × gain_lin(θ,φ) × sin(θ) × dθ × dφ`
   - 对全空间求和，与各向同性天线的总功率之比即算得MEG（线性，可转dB）
2. 同时计算 MEG 在各极化分量上的分解（co-pol MEG, cross-pol MEG）
3. `core/result/ExportChannel.cpp`：新增 MEG 字段导出

**验收条件**：
- 各向同性天线+均匀APS → MEG ≈ 0 dBi
- 定向天线（主瓣对准强APS方向） → MEG > 各向同性天线
- 天线主瓣偏离强APS方向 → MEG < 标称峰值增益

## 4. 科学分析方法指南

### 4.1 核心分析框架：天线-信道解耦分析

**理论基础**（Steinbauer 2001; Rasp 2024）：

```
物理传播信道（天线无关）  ×  Tx天线响应  ×  Rx天线响应  =  可观测信道
         ↓                      ↓              ↓              ↓
    GeometricPath            G_tx(θ,φ)     G_rx(θ,φ)     CIR/PDP/APS
    (geometry only)       E_tx(θ,φ)      E_rx(θ,φ)     /XPR/MEG
```

**分析策略**：
1. 固定几何路径集（同一场景、同一SBR参数）
2. 仅切换天线模型（Ideal → Dipole → Patch → Horn）
3. 观察可观测信道指标的变化
4. 将差异归因于天线方向图、极化选择性、姿态的具体特征

### 4.2 E1 天线模型对比——如何分析

#### 4.2.1 PDP 分析

**生成图表**：
- 四天线模型的PDP叠加图（同一delay轴，dBm功率轴）
- 标注LoS路径和各反射路径的delay位置

**分析要点**：
1. **总接收功率差异**：不同天线因方向图差异导致的总接收功率差异（dB），与各自峰值增益的差异是否一致？若不一致，说明多径到达角分布使天线"有效增益"偏离标称峰值增益
2. **PDP形状变化**：不同天线对不同delay路径的相对加权——方向性天线（patch/horn）的主瓣方向若对准主要反射路径到达方向，该路径会被增强；若偏离，会被抑制
3. **LoS与多径功率比**：定向天线可能使LoS/多径功率比增大（主瓣增强LoS，旁瓣抑制多径）
4. **与Ideal各向同性天线的差异**：以Ideal天线为基准，计算其他天线在每个delay bin的相对增益/衰减

**论文表述示例**：
> 在相同的几何传播路径集下，当接收天线从各向同性切换为定向贴片天线（主瓣朝向Tx方向）时，总接收功率增加了 X dB，小于贴片天线的标称峰值增益 Y dBi。这一差异源于入射多径的角度分布——仅 Z% 的入射功率集中在贴片天线主瓣的3dB波束宽度内，其余功率从旁瓣方向到达并被衰减。

#### 4.2.2 APS 分析

**生成图表**：

- 四天线模型的AoA二维(theta-phi)功率谱热图（2×2 subplot）
- 叠加天线方向图等高线（以天线局部坐标系为参考）

**分析要点**：
1. **APS"滤波"效应**：传播信道的APS（天线无关）被天线方向图"调制"后，观测到的APS发生形变——这是MEG公式的直观体现
2. **主瓣方向偏离**：若天线主瓣未对准主要能量到达方向，APS热图中心（主瓣方向）的功率反而低于偏离方向
3. **极化引起的APS不对称**：对具有非对称极化方向图的天线（如patch天线的交叉极化在某些方向增强），AoA φ维度出现不对称性
4. **APS相似性量化**：计算两天线模型APS的相关系数或KL散度，给出数值化的"天线模型差异引起的角谱差异"度量

#### 4.2.3 XPR/XPD 分析

**生成图表**：
- 四天线模型的XPR CDF曲线叠加图
- 路径级XPR vs AoA scatter图（每个天线一个子图）

**分析要点**：
1. **固定极化vs Jones极化**：若天线使用固定极化矢量，XPR仅取决于入射电场极化状态和固定极化矢量的匹配；若使用逐角度Jones极化，XPR还取决于到达角——到达角变化时，天线的"有效极化方向"也在变化
2. **天线类型的XPR特征**：偶极子（线极化）的XPR分布与喇叭（可能有显著交叉极化分量）的XPR分布有何不同？
3. **XPR与到达角的相关性**：高XPR路径是否集中在特定AoA方向？低XPR路径是否对应天线的交叉极化增强方向？
4. **与3GPP参考值对比**：3GPP TR 38.901 中XPR参数通常为6-15dB/簇——仿真得到的XPR分布与此是否可比？差异如何解释？

**论文表述示例**：
> 采用逐角度Jones极化模型时，XPR的中位数从固定极化模型的 A dB 变化为 B dB，且分布的标准差增大了 C dB。这反映了天线在不同辐射方向上极化纯度的变化——在主瓣方向，天线的交叉极化分量通常低于-20 dB；但在偏离主瓣60°以上的方向，交叉极化可能升至-10 dB，导致大角度到达路径的XPR显著降低。

#### 4.2.4 MEG 分析

**生成图表**：
- 四天线模型的MEG对比柱状图（标称峰值增益 vs 计算MEG）
- MEG vs 天线主瓣-APS主方向夹角图

**分析要点**：
1. **MEG与峰值增益的差异**：MEG反映了天线在实际多径环境中的"有效增益"。高峰值增益的天线（如horn, ~15 dBi）在室内多径环境中，MEG可能远低于峰值增益（如仅8 dBi），因为它只对窄角度范围内的入射波高效接收
2. **MEG的极化分解**：co-pol MEG和cross-pol MEG各自的分量，揭示接收功率中有多少来自"匹配极化"，多少来自"交叉极化泄漏"
3. **与Kalliola 2002的测量结果对比**：该文献报告了移动终端天线在不同传播环境中的MEG测量值（室外-3到-1 dBi，室内-5到-2 dBi）——仿真得到的MEG值与此范围是否可比？

### 4.3 E2 姿态敏感性——如何分析

**生成图表**：
- 总接收功率 vs yaw角 曲线（含误差棒或ray_count变化范围）
- XPR均值/中位数 vs yaw角 曲线
- 特定yaw角下的APS热图（选3-4个代表性角度：0°, 45°, 90°, 135°, 180°）

**分析要点**：
1. **功率-姿态曲线的不对称性**：若场景非对称（如Tx不在Rx正前方），功率-姿态曲线应呈现非对称形状，峰值出现在主瓣对准主要入射方向的角度
2. **主径功率 vs 总功率**：主径（通常LoS或最强单反射径）功率随姿态变化剧烈（窄波束×角度敏感），而总多径功率变化平缓（多径从各方向到达，角度平均化）
3. **XPR的姿态依赖性**：线极化天线在旋转时，其极化方向在世界坐标中旋转——这直接改变XPR
4. **临界角度**：是否存在某个姿态角使接收功率骤降（如主瓣的零陷方向恰好对准LoS到达方向）？

### 4.4 收敛性与敏感性分析——如何写

**C1 射线数收敛性**：
- 以最高射线数（如500k）的结果为参考值
- 计算其他射线数的总功率相对误差、PDP峰值位置偏差、XPR均值偏差
- 若误差在50k→200k区间内收敛至<1 dB，可声称"在选定射线数参数下，信道指标进入稳定区间"
- **不要声称"路径已穷尽"或"结果已收敛"**——只声称"主要信道指标对射线数的敏感性低于1 dB"

**C2/C3 敏感性**：
- 类似方法，以实验参数为自变量，信道指标为因变量
- 用敏感性曲线证明参数选取不是任意的，而是在"结果稳定、计算成本可控"的区间
